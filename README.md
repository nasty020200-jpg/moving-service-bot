# Бот грузоперевозок для Vercel

Telegram бот для принятия заявок на грузоперевозки с webhook архитектурой.

## Быстрый деплой (через веб-интерфейс)

1. Создайте репозиторий на GitHub
2. Загрузите все файлы из папки `vercel_bot`
3. Зайдите на https://vercel.com и подключите GitHub
4. Импортируйте репозиторий и нажмите Deploy
5. Получите URL и установите webhook:
   ```
   https://api.telegram.org/bot8367829067:AAFCZZji7PUZPCUmNcFRi-1E958bnjxVNpk/setWebhook?url=https://ВАШ-URL.vercel.app/
   ```

Подробная инструкция в файле `vercel_deployment_guide.md`

## Настройка администратора

После деплоя отправьте команду `/setadmin` боту с аккаунта @PereezdBatumiGE
